About This Server

Learn about the technology powering this web server

This is a custom HTTP server written in C++ using Windows Sockets (Winsock), designed to demonstrate low-level network programming and HTTP protocol implementation.

Technical Details

Language
C++
Socket API
Winsock2
Protocol
HTTP/1.1
Architecture
Single-threaded with blocking I/O

Server Components

Server.cpp

Main server loop and client connection handling. Manages the core server lifecycle and accepts incoming connections.

Router.cpp

URL routing and response generation. Maps incoming requests to appropriate handlers and generates HTTP responses.

HttpRequest.cpp

Data structure to hold HTTP request information including headers, method, and body.

HttpResponse.cpp

Data structure to hold HTTP response information including status codes, headers, and content.

Server Capabilities

HTTP Methods

GET, POST, PUT, DELETE

Status Codes

200, 404, 500, etc.

MIME Types

HTML, CSS, JS, JSON

About This Server

This is a custom HTTP server written in C++ using Windows Sockets (Winsock).

Technical Details:

  • Language: C++
  • Socket API: Winsock2
  • Protocol: HTTP/1.1
  • Architecture: Single-threaded with blocking I/O

Components:

  • Server.cpp: Main server loop and client handling
  • httpDecoder.cpp: HTTP request/response parsing
  • route.cpp: URL routing and response generation
  • httpRequestResponse.h: Data structures and function declarations
